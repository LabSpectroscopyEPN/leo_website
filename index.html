Publicar en GitHub

¿Cómo funciona?
Al pulsar "Publicar", el panel sube contenido.json a tu repositorio. Las imágenes ya se subieron automáticamente al guardar cada entrada. GitHub Pages actualiza el sitio en ~30 segundos.

Configuración del token (solo una vez):
Ve a GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Crea un token con permiso Contents: Read and write solo para este repositorio.
Sin configurar

Se guarda solo en este navegador (localStorage). Nunca se envía a servidores externos.
